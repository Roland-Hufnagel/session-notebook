# Git Commands:

- git status: Show status of branch
- git add: Add file or folder to staged
- git commit -m \<message>: Commitment to git
- git branch: List your branches
- git branch -a: List all branches
- git branch \<branchname>: create a new branch
- git branch -d: delete a branch
- git switch -c \<branchname>: Create a new branch with the switch comman
- git switch: switch branches
- git checkout: Another way to switch branches
